## Study Strategy

**For a Node.js / backend engineering role:**
1. Start with `node-interview-prep/` — use the README there for a structured study plan
2. Use `dsa/` in parallel for coding round preparation
3. End with the cheat sheet: `node-interview-prep/12-interview-practice/00-cheat-sheet/`

**For a pure DSA / FAANG-style round:**
1. Work through `dsa/` by topic (arrays → trees → graphs → DP)
2. Practice the patterns before the specific problems
3. Time yourself: easy = 15 min, medium = 25 min, hard = 40 min
